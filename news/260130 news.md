## 1. Google Genie 3, 텍스트 프롬프트로 실시간 인터랙티브 월드 생성

**Google DeepMind, 세계 최초 실시간 범용 월드 모델 공개...AGI 향한 핵심 이정표**

Google DeepMind가 1월 29일 실험적 연구 프로토타입 **'Project Genie'를 통해 Genie 3 월드 모델을 일반 사용자에게 공개**했다. Genie 3는 텍스트나 이미지 프롬프트만으로 **720p 해상도, 24fps의 실시간 인터랙티브 3D 환경**을 생성할 수 있는 범용 월드 모델로, 사용자가 생성된 세계 안에서 직접 탐험하고 물체와 상호작용할 수 있다. DeepMind 연구 디렉터 Shlomi Fruchter는 "Genie 3는 특정 환경에 국한되지 않고 포토리얼리스틱한 세계부터 상상 속 세계까지 모두 생성할 수 있는 **최초의 실시간 범용 월드 모델**"이라고 설명했다.

Genie 3는 2025년 8월 연구 프리뷰로 공개된 후 내부 테스터들의 긍정적 반응을 얻어 이번에 Google AI Ultra 구독자(월 249.99달러) 대상으로 미국에서 먼저 출시됐다. 이전 버전 Genie 2가 10~20초 분량의 환경만 생성할 수 있었던 것과 달리, Genie 3는 **수 분간 물리적으로 일관된 환경을 유지**하며 1분 전에 방문했던 장소도 기억한다. DeepMind는 이 기술이 게임과 교육을 넘어 자율주행차 훈련, 로봇공학, 역사적 시대 탐험 등에 활용될 수 있으며, **AGI(범용 인공지능) 달성을 위한 핵심 발판**이 될 것이라고 강조했다.

## 2. Claude Code, MCP 서버와 Skill 결합으로 워크플로우 자동화 극대화

**Anthropic, MCP-Skill 통합 가이드 공개..."AI 에이전트의 핵심은 연결+전문성"**

Anthropic이 2025년 12월 Claude Code 사용자를 위한 **MCP(Model Context Protocol) 서버와 Skill 통합 활용 가이드**를 공개했다. MCP는 Claude를 GitHub, Notion, Salesforce 등 외부 도구에 연결하는 표준 프로토콜이고, Skill은 해당 연결을 효과적으로 활용하는 방법을 가르치는 지침 패키지다. Anthropic은 "MCP가 연결(connectivity)을 담당한다면, **Skill은 그 연결을 신뢰할 수 있는 결과로 바꾸는 전문성(expertise)**을 제공한다"고 설명했다.

실제 활용 사례로 Notion MCP 서버 연결만으로는 Claude가 문서를 검색할 수 있지만, 미팅 준비용 Skill을 추가하면 **어떤 페이지를 참조하고, 어떤 형식으로 문서를 작성하며, 팀 표준을 어떻게 적용할지**까지 자동으로 처리한다. 한 개발자 커뮤니티 글에서는 "Skill 없이는 Claude가 매번 추측해야 하지만, Skill이 있으면 **팀의 플레이북을 그대로 따른다**"고 평가했다. Anthropic은 2025년 10월 Agent Skills를 크로스 플랫폼 호환 표준으로 공개하며 MCP 생태계 확장에 박차를 가하고 있다.

## 3. Moltbot 개발자 Peter Steinberger, "AI가 쓴 코드 읽지 않고 배포"

**1월 한 달 6,600건 커밋...GitHub 역대 최고 스타 성장 기록 중인 AI 에이전트**

오스트리아 개발자 Peter Steinberger가 만든 개인용 AI 에이전트 **Moltbot(구 Clawdbot)이 GitHub 역사상 가장 빠른 스타 성장을 기록**하며 주목받고 있다. PDF 소프트웨어 기업 PSPDFKit(현 Nutrient)를 창업해 성공적으로 매각한 Steinberger는 2025년 11월 이 프로젝트를 공개한 후, 1월 한 달 동안만 **6,600건 이상의 커밋을 혼자서 기록**했다. 그는 인터뷰에서 "나는 **AI가 작성한 코드를 읽지 않고 배포한다**"며 '바이브 코딩(vibe coding)'의 극단적 사례를 보여줬다.

Moltbot은 사용자의 기기에서 로컬로 실행되며 WhatsApp, Telegram, Slack, Discord 등 메시징 앱과 연동해 **예약, 이메일 관리, 파일 정리 등을 자율적으로 수행**한다. 프로젝트는 1월 27일 Anthropic의 상표권 요청으로 Clawdbot에서 Moltbot으로 이름을 변경했는데, 이 과정에서 GitHub와 X 계정 이름이 **10초 만에 암호화폐 사기꾼들에게 탈취**되는 해프닝이 벌어지기도 했다. 프로젝트의 인기로 Mac mini 판매량이 급증했다는 보도가 나올 정도로, **로컬 AI 에이전트에 대한 시장의 관심이 폭발적**임을 보여준다.

## 4. NVIDIA Parakeet TDT 0.6B, 60분 오디오를 1초에 전사

**6억 파라미터로 25개 유럽어 지원...오픈소스 ASR 리더보드 1위**

NVIDIA가 **Parakeet TDT 0.6B v3**를 공개하며 오픈소스 음성 인식(ASR) 분야에서 새로운 기준을 제시했다. 6억 개 파라미터의 이 모델은 **60분 분량의 오디오를 단 1초에 전사**할 수 있으며, Hugging Face Open ASR 리더보드에서 **6.05% 단어 오류율(WER)로 1위를 기록**했다. v3 버전은 기존 영어 전용에서 한국어를 제외한 **25개 유럽 언어로 지원을 확대**하고 자동 언어 감지 기능을 추가했다.

이 모델은 FastConformer 인코더와 TDT(Token-and-Duration Transducer) 디코더를 결합한 아키텍처로, **최대 24분 오디오를 풀 어텐션으로, 3시간까지 로컬 어텐션으로 처리**할 수 있다. NVIDIA는 Granary 다국어 데이터셋과 자체 NeMo ASR Set 3.0의 10,000시간 휴먼 전사 데이터로 훈련했다고 밝혔다. **CC-BY-4.0 라이선스**로 공개돼 상용화가 가능하며, 한 개발자는 Ryzen 9 홈랩 서버에서 Claude Code로 MacOS 전사 앱을 만들어 "**거의 실시간에 가까운 전사**"를 경험했다고 공유했다.

## 5. Anthropic, 2029년 매출 1,480억 달러 전망..."OpenAI보다 2년 먼저 흑자"

## 6. Anthropic CEO "6~12개월 내 AI가 소프트웨어 엔지니어링 전 과정 담당"

Anthropic CEO Dario Amodei가 1월 23일 다보스 세계경제포럼에서 "**AI가 6~12개월 내 소프트웨어 엔지니어가 하는 대부분, 어쩌면 전부를 end-to-end로 처리**할 수 있게 될 것"이라고 예측해 기술 업계에 충격을 줬다. 그는 "Anthropic 내부에 '나는 더 이상 코드를 직접 쓰지 않는다. 모델이 코드를 쓰고 나는 편집만 한다'고 말하는 엔지니어들이 있다"며 **실제 업무 방식의 변화**를 공개했다.

## OpenAI, 내부 AI 데이터 에이전트 '케플러' 공개…일일 600PB 데이터 처리

OpenAI가 내부 운영의 핵심으로 자리 잡은 **AI 데이터 분석 플랫폼 '케플러(Kepler)'를 공개**했다. 케플러는 GPT-5와 o-시리즈 모델의 추론 능력을 활용해, 3,500명 이상의 직원이 기술적 배경과 관계없이 **7만 개 이상의 내부 데이터셋을 자연어로 쿼리하고 분석**할 수 있게 해주는 자율형 데이터 분석가 역할을 수행한다. 이 시스템은 하루 약 600페타바이트의 신규 데이터를 생성하는 OpenAI의 복잡한 데이터 인프라를 관리한다.

케플러의 핵심 기술은 **메타데이터 기반 RAG(검색 증강 생성)**다. 원시 데이터가 아닌 수만 개 데이터셋의 설명과 스키마를 인덱싱해, 페타바이트 규모의 로그를 스캔하는 계산 부담 없이 정보 위치를 파악한다. 또한 'Codex 테스트'라는 자동 검증 레이어를 통해 생성된 SQL이나 Python 코드의 구문과 논리를 사전 검증함으로써 **AI 환각(hallucination) 문제를 완화**한다. 업계에서는 케플러를 차세대 '인지 기업(Cognitive Enterprise)'의 청사진으로 평가하고 있다.
